 <h2>stack.ToMap()</h2>

 `stack.ToMap() (newMap map[any]any)`

```
 Creates a new map whose keys and values correspond to the cards in `stack`

 @examples
 | MakeStack([]int {1, 2, 3}, []string {"a", "b", "c"}).ToMap() => map[any]any {1:"a", 2:"b", 3:"c"} // in any order
```
